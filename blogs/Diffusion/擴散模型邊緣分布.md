`当前时刻的score function可以通过条件分布来计算`
$$
\nabla \log p_t(x_t) = \mathbb{E}_{x_0 \sim p(x_0|x_t)}[\nabla \log p(x_t|x_0)
$$
这个等式揭示了一个深刻的关系: `当前时刻的score function可以通过条件分布来计算`。让我从几个角度帮你理解:

## 1. 贝叶斯视角

首先回顾贝叶斯公式: $$p(x_t|x_0) p(x_0) = p(x_0|x_t) p(x_t)$$

两边取对数再对$x_t$求导: $$\nabla_{x_t} \log p(x_t|x_0) + \nabla_{x_t} \log p(x_0) = \nabla_{x_t} \log p(x_0|x_t) + \nabla_{x_t} \log p(x_t)$$

因为$p(x_0)$不依赖$x_t$,所以第二项为0,移项得: $$\nabla_{x_t} \log p(x_t) = \nabla_{x_t} \log p(x_t|x_0) - \nabla_{x_t} \log p(x_0|x_t)$$

对$x_0$取期望(在$p(x_0|x_t)$下): $$\nabla \log p_t(x_t) = \mathbb{E}_{x_0 \sim p(x_0|x_t)}[\nabla \log p(x_t|x_0)]$$

## 2. 直觉理解

**关键洞察**: 边缘分布的score是条件分布score的期望

- **左边** $\nabla \log p_t(x_t)$: 在所有可能的干净数据$x_0$下,$x_t$的总体"梯度方向"
- **右边**: 对每个可能的$x_0$来源,计算$\nabla \log p(x_t|x_0)$(前向过程的score),然后按后验$p(x_0|x_t)$加权平均

**物理类比**: 想象$x_t$是一个观测到的噪声数据点

- 它可能来自多个不同的干净数据$x_0$
- 每个$x_0$都会"拉"$x_t$朝某个方向(这就是$\nabla \log p(x_t|x_0)$)
- 最终的score是所有这些"拉力"的加权平均,权重就是"这个$x_0$真的是源头"的概率

## 3. 扩散模型中的应用

在扩散模型中,$p(x_t|x_0)$通常是高斯分布,所以: $$\nabla \log p(x_t|x_0) = -\frac{x_t - \mu_t(x_0)}{\sigma_t^2}$$

这意味着: $$\nabla \log p_t(x_t) = \mathbb{E}_{x_0}[-\frac{x_t - \mu_t(x_0)}{\sigma_t^2}] = -\frac{x_t - \mathbb{E}[\mu_t(x_0)]}{\sigma_t^2}$$

**所以score function实际上指向条件均值的期望方向!**

这就是为什么训练去噪模型$\epsilon_\theta$等价于学习score function的原因。
